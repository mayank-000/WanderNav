# **App Name**: WanderNav

## Core Features:

- Destination Search: Users can search for destinations using keywords. This search will surface available places to visit within the area.
- AI-Powered Trip Planner: AI tool analyzes the travel blogs in the database. Based on user inputs, such as budget, travel dates and points of interests, AI reasons through these and other attributes to create a possible travel plan. 
- Blog Post Display: Showcases individual blog posts with a title, images, text, and relevant metadata like the date published and author.
- Category/Tag Filtering: Filter travel blogs based on categories, regions, or travel styles (e.g., adventure, budget-friendly, luxury).
- User Account: Users can sign in or register in the app to save favorite blogs and configure their search preferences
- Image Gallery: Display beautiful pictures of the visiting place

## Style Guidelines:

- Primary color: HSL(45, 80%, 50%) which translates to a vivid golden yellow (#F5B740) to represent adventure and exploration.
- Background color: Desaturated yellow at HSL(45, 20%, 90%), or #F4F2EC. Light scheme.
- Accent color: Analogous color at HSL(15, 60%, 60%), giving a light coral shade (#E07A5F) for interactive elements.
- Font pairing: 'Playfair' (serif) for headlines and 'PT Sans' (sans-serif) for body text, providing an elegant yet readable feel.
- Use a set of minimalist and clear icons, related to traveling (a plane, a hotel, etc.)
- Clean, modern layout with an emphasis on high-quality photography. Use a grid system to present content in an organized and visually appealing manner.
- Smooth transitions and subtle animations to enhance user experience, like a loading animation, or smooth scrolling between pages.
